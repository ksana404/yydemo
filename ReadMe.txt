=模块分为七个模块:
后台管理系统: yy-manager
Web前台: yy-web 
通用层: yy-common
单点登录模块: yy-sso
购物车模块: yy-cart
订单模块: yy-order
swagger演示模块: yy-swagger

=启动步骤
 --1.启动Zookeeper集群
 --2.启动Redis集群
 --3.启动Nginx
 路径: H:\desktop\dev-others\nginx-1.15.10
 指令: 	1).路径栏 CMD回车  
 		2)输入 start ngnix 回车! 
 		3)在资源管理器进程项,按住英文n,查看ngnix出现两个图标,则说明启动,启动失败dos窗口会报错.
 
 4.启动后台管理 yy-manager和其他模块,Web前台: yy-web 模块最后一个启动.
 5.浏览器访问
 后台:http://manage.yy.com/
 前台:http://www.yy.com/
 
=功能简介
	后台:商品列表的查询,商品新增,修改,删除,上架下架.
	
	前台:商城主页访问,
		用户注册,登录,退出.
		商品详情查看
		商品购物车添加,购物车商品 数量修改,删除.
		订单详情预览,订单提交,订单提交结果页显示.
